# Proyecto Red Social en React

## Descripción
Este proyecto es una aplicación tipo red social desarrollada con React. Permite iniciar sesión con diferentes usuarios, ver un feed de publicaciones, enviar mensajes tipo chat entre usuarios y visualizar perfiles con información básica.

El objetivo del proyecto es simular el comportamiento de una red social usando React, React Router y almacenamiento local (LocalStorage).

## Funcionalidades

- Inicio de sesión con usuarios predefinidos
- Feed de publicaciones tipo red social
- Creación de posts
- Sistema de chat entre usuarios (tipo WhatsApp)
- Perfil de usuario con estadísticas
- Exploración de usuarios
- Persistencia de datos con LocalStorage

## Instalación y ejecución

1. Clonar el repositorio:
git clone <url-del-repositorio>

2. Entrar al proyecto:
cd nombre-del-proyecto

3. Instalar dependencias:
npm install

4. Ejecutar el proyecto:
npm run dev

## Notas del proyecto

- El sistema de autenticación es simulado mediante LocalStorage.
- Los chats se guardan localmente en el navegador.
- Este proyecto no utiliza backend real.
